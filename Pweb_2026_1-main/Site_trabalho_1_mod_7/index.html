Minecraft é um jogo de aventura sandbox em 3D desenvolvido pela Mojang Studios em que o jogador interage com um ambiente tridimensional totalmente modificável feito de blocos e entidades. Sua jogabilidade diversa permite que o jogador escolha a forma como joga, com inúmeras possibilidades. Há várias edições do Minecraft que recebem manutenção ativa, sendo elas a:

Edição Java para Windows, macOS e Linux;

Minecraft Education, uma variante da Edição Bedrock para salas de aula.

Edição China, um lançamento localizado de ambas as edições para a China continental.
